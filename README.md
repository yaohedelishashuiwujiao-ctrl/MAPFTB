# MAPFTB

MAPFTB 是一个基于公开数据的个人汽车底盘自主研究工具。

系统根据研究问题建立 Research Checklist，自主查询和收集证据，识别数据缺口，对 AI 生成的 Claim 执行来源、版本、单位和代码级验证，并输出可审计的竞品研究结果。

示例输入：

> 调研大型纯电 MPV 竞品的 H 臂、转向器前置后悬架方案，分析方案差异和可借鉴之处。

## 核心方向

- 数据缺口驱动的自主研究 Agent
- SQL、搜索、RAG 和 Python 计算工具路由
- 类型化 Claim Verification Engine
- 多来源冲突与车型版本治理
- Checkpoint、人工审核、失败恢复和预算控制
- 自动评测、Trace、Token、费用和延迟统计
- 后续扩展底盘零部件视觉检查与持续学习

## 开发路线

```text
V0 可行性基线
-> V1 可信结构化查询与计算工具
-> V2 数据缺口驱动的研究 Agent
-> V3 Claim Verification 与生产可靠性
-> V4 底盘视觉检查器 PoC
-> V5 持续学习视觉系统
-> V6 证据驱动研究汇报
```

`V3` 是第一阶段求职展示版本。视觉能力属于后续高风险扩展，不阻塞一级 Agent 完成。

## 项目边界

- 仅使用有权访问的公开数据，不使用或公开公司内部数据。
- 项目是个人研究工具，不代表任何公司的正式项目。
- 不将第三方模型或开源能力描述为自研算法。
- 高风险工程结论默认需要人工审核。
